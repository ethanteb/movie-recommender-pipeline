# Movie Recommendation Engine

movie-recommender-pipeline is a project aimed at demonstrating Python programming skills

## Installation

```bash
git clone https://github.com/ethanteb/movie-recommender-pipeline
cd project

python -m venv venv
source venv/bin/activate

pip install -r requirements.txt
python main.py
```

## Usage

When running main.py enter the name of a movie that you would like similar recommendations to. The program will print the top five recommendations of similar movies based on similarity and audience ratings.
